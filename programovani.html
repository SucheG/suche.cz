Python a Django

Python je programovací jazyk, který má přehlednou a uživatelsky přívětivou syntaxi. Podporuje mnoho programovacích stylů a těší se velké popularitě. Často se učí na školách a v praxi se uplatní ve strojovém učení, práci s texty, tvorbě webů, datové analýze. Často bývá jako skriptovací jazyk u různých systému (ať už hardware nebo sw produkty). Můžete v něm udělat téměř vše.

Django je propracovaný webový framework pro tvorbu webových aplikací postavený na Pythonu. Mnoho věcí je tam vyřešených a vývojář se může soustředit na samotnou logiku aplikace.

HTML, CSS, JS, Bulma

HTML, CSS a JavaScript v dnešní době internetu patří neodmyslitelně k sobě. Neexistuje moderní web, kterému by chyběla jedna z těchto součástí. HTML je značkovací jazyk, který pomocí tagů definuje strukturu webu. CSS jsou pak pravidla pro vzhled, přechody a animace. JavaScript je pak funkční a logická část, pomocí kterého se dá ovládat vše, co se na stránce děje.

Vztah webu a jeho součástí: web se skládá z více stránek (pokud to není jednostránkový web). Stránky jsou navzájem propojené odkazy a každá stránka je definovaná HTML kódem. Pro celý web by pak měl existovat jednotný styl definovaný pomocí CSS a JavaScript, který vykonává potřebné chování.

Bulma je CSS framework vytvořený pro tvorbu vzhledu webu, jsou tam definované různé komponenty a chování, které si lze podle vlastního uvážení přenastavit. Tento framework je responzivní - to znamená, že se dokáže přizpůsobit různým velikostem displaye.

První lekce

Webové stránky v dnešní době se skládají z HTML, JS a CSS a mediálních souborů jako jsou obrázky a videa.

do HTML se zapisuje struktura - nadpis, odstavec, seznamy apod.
CSS definuje vzhled: např barva a velikost písma, pozice
JavaSript (zkratka JS) se používá pro logiku - např. co se má stát když se stiskne tlačítko

Otevři si konzoli vývojáře (F12 pro Chrome) a záložka Console

V tomto úkolu si vytvoříme tlačítko, které zavře modální okno. Do konzole napiš následující příkazy (šedě jsou poznámky, ty kopírovat nemusíš).

// vytvoříme element tlačítko
let button = document.createElement('button');
// vybereme prvek do kterého tlačítko vložíme
let parent = document.querySelector('#pro-modal .notification');
// přidáme tlačítko do prvku, už je vidět, ale jen malý čtverec
parent.appendChild(button);
// změníme tlačítku text
button.textContent = 'Zavřít';
// a přidáme třídu pro CSS styl
button.classList.add('button');
// zkus tkačítko stisknout - nic nebude dělat
// nastavíme funkci pro zavření okna, nyní už zavírá okno
button.onclick = function() {
  hideModal('pro-modal');
}
